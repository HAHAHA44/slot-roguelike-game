# Balance Checklist

- Cap copy and amplify chains before they create infinite settlement loops.
- Cap permanent growth so a single line cannot scale forever without board pressure.
- Preserve at least one stable route through crisis contracts at every difficulty tier.
- Keep wild tokens below the best-in-slot baseline for specialized tag builds.
- Treat multi-cell tokens as situational tools, not the default strongest option.
- Review endless target ramps against average turn output before raising the step size.
- Keep temporary board expansions opt-in and anomaly-scoped so base 5x5 balance stays intact.
